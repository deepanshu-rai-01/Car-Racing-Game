# 🚗 Car Racing Game

## 📌 Description
A simple 2D Car Racing Game built using Python and Pygame. The player controls a car and avoids oncoming enemy cars while trying to achieve the highest possible score.

## ✨ Features
- Player-controlled car
- Random enemy car generation
- Collision detection
- Score tracking
- Increasing difficulty
- Smooth gameplay

## 🛠 Technologies Used
- Python
- Pygame

## ▶️ How to Run

```bash
pip install pygame
python car_racing_game.py
```

## 🎮 Controls
- ⬅ Left Arrow → Move Left
- ➡ Right Arrow → Move Right
